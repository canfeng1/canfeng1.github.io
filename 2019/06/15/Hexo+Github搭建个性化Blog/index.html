搭建Hexo博客的过程中遇到的问题和解决办法

  • 1 输入hexo -v，出现错误提示：zsh: command not found: hexo,输入npm install hexo-cli -g 来安装hexo，发现还是提示zsh: command not found: hexo,可能权限不够，在命令前加上sudo，用管理员身份成功安装,输入sudo npm install hexo-cli -g，安装hexo成功。

  • 2 对 Hexo 根目录 _config.yml 的修改，没有反应，需要重启本地服务器后才能预览效果。

  • 3 运行git clone https://github.com/iissnan/hexo-theme-next.git ，安装next，运行hexo相关指令就会出现下列错误提示：

  • 4 删掉当前next版本v5.1.4，更换成最新的next主题版本v7.1.2，运行git clone https://github.com/theme-next/hexo-theme-next.git，安装成功，提示消失。

  • 5 需要删掉用命令新建的文章或页面时，只需要进入 Hexo 根目录下的 source 文件夹，删除对应文件或文件夹即可。
